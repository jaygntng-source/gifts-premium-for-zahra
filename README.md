# Hadiah Digital Premium - Anniversary Zahra & Fikri

Website galeri interaktif romantis ini siap digunakan untuk hadiah anniversary yang eksklusif, mewah, dan emosional. Proyek ini sudah dirancang otomatis agar tidak butuh coding tambahan. 

## Cara Menggunakan (Hanya 3 Langkah Mudah!)

### 1. Masukkan Lagu Kesukaan Kalian
1. Siapkan file lagu berformat `.mp3`.
2. Ubah nama file lagu kamu menjadi **`lagu.mp3`**.
3. Letakkan file tersebut di dalam folder: `assets/audio/` (Timpa file lama jika ada).

### 2. Masukkan Foto Kenangan (Rasio 4:3)
1. Siapkan 8 foto horizontal (Mendatar).
2. Ubah namanya berurutan menjadi: `photo1.jpg`, `photo2.jpg`, sampai `photo8.jpg`.
3. Letakkan semua foto di dalam folder: `assets/images/memories/`

### 3. Masukkan Screenshot Chat (Rasio 3:4)
1. Siapkan 4 screenshot chat vertikal (Memanjang ke bawah).
2. Ubah namanya menjadi: `chat1.jpg`, `chat2.jpg`, `chat3.jpg`, `chat4.jpg`.
3. Letakkan semua chat di dalam folder: `assets/images/chats/`

## Cara Mempublikasikan Secara Gratis (Upload ke GitHub Pages)
1. Buat akun di [GitHub](https://github.com/).
2. Buat repository baru, beri nama bebas (misal: `untuk-zahra`).
3. Upload seluruh file dan folder ini ke repository tersebut.
4. Pergi ke tab **Settings** di repository kamu.
5. Klik menu **Pages** (ada di menu sebelah kiri).
6. Pada bagian *Source*, pilih branch `main` atau `master`, lalu klik **Save**.
7. Tunggu sekitar 2-5 menit, GitHub akan memberikan link website kamu yang sudah jadi.
8. Kirim link tersebut ke Zahra! Selamat Anniversary!
