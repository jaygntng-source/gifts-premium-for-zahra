# Premium Digital Anniversary Experience Website

Situs web eksklusif bertema *luxury romance storytelling* untuk merayakan perjalanan cinta dua tahun. Dikembangkan menggunakan arsitektur Vanilla JavaScript berkinerja tinggi, CSS modern, serta animasi premium berbasis GSAP dan Swiper.js.

## Fitur Utama

1. **Premium Loading System**: Progress bar elegan disertai kurasi kutipan romantis acak.
2. **Interactive Math Logic Game**: Gerbang interaktif bertema *Unlock Our Memories* untuk membuka sisa konten cerita.
3. **Pinterest Masonry Polaroid Gallery**: Tampilan galeri foto dengan rotasi kasual asimetris menyerupai kertas fisik asli.
4. **Interactive Love Letter**: Amplop digital interaktif dengan visual lilin segel (*wax seal*) dan tulisan tangan beranimasi.
5. **Dynamic Parallax Timeline**: Garis waktu perjalanan cinta yang responsif dan interaktif.
6. **Luxury Chat Carousel**: Slider berkinerja tinggi untuk menampilkan potongan tangkapan layar percakapan manis.
7. **3D Unboxing Gift Card**: Kejutan interaktif kupon penukaran berhias refleksi holografik.
8. **Starlight Final Wish**: Langit malam interaktif berbasis HTML5 Canvas untuk mengirimkan harapan.
9. **Floating Vinyl Music Control**: Pemutar musik latar belakang mengambang dengan visual piringan hitam berputar.
